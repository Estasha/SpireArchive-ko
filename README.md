# SpireArchive

슬레이 더 스파이어2의 모든 정보를 담은 종합 데이터베이스 웹사이트

## 프로젝트 개요

SpireArchive는 슬레이 더 스파이어2 게임의 모든 정보를 체계적으로 정리하여 제공하는 팬 제작 웹사이트입니다. 한국인 게이머들을 위해 최적화된 인터페이스와 상세한 정보를 제공합니다.

## 주요 기능

### 🎭 캐릭터 정보
- **아이언클래드**: 강력한 전사, 방어와 공격의 밸런스
- **사일런트**: 은밀한 암살자, 독과 카드 조합
- **리젠트**: 새로운 지배자, 지배와 제어 능력
- **네크로바인더**: 사령술사, 소환과 사령술
- **디펙트**: 자동화된 기계, 오브 활용

### 🃏 카드 정보
- **캐릭터별 카드**: 각 캐릭터의 고유 카드 상세 정보
- **무색 카드**: 모든 캐릭터가 사용할 수 있는 범용 카드
- **카드 필터링**: 타입, 비용, 희귀도별 필터링
- **상세 정보**: 카드 효과, 전략, 조합 정보

### 💎 유물 정보
- **일반/희귀/영웅 유물**: 각 등급별 유물 상세 정보
- **보스 유물**: 보스 보상 유물 정보
- **이벤트 유물**: 특수 이벤트 유물 정보
- **조합 추천**: 유물 조합별 시너지 정보

### 🧪 포션 정보
- **일반/희귀/고급 포션**: 각 등급별 포션 상세 정보
- **특수 포션**: 희귀하고 강력한 특수 포션 정보
- **효과별 분류**: 공격, 방어, 유틸리티, 회복, 디버프
- **사용 전략**: 최적의 포션 사용 타이밍과 조합

### 🎲 이벤트 정보
- **층별 이벤트**: 엑소르디움, 도시, 더 비욘드, 더 엔딩
- **선택지별 결과**: 모든 선택지와 그에 따른 결과
- **특수 이벤트**: 희귀하고 특별한 이벤트 정보

### 👹 적 정보
- **층별 적 정보**: 각 층의 일반, 엘리트, 보스 적
- **공격 패턴**: 적의 공격 패턴과 행동 분석
- **대처 전략**: 각 적에 대한 최적의 대처 전략
- **난이도 분류**: 적의 난이도별 분류

### ⚡ 고대의 존재 정보
- **공격/스킬/파워 고대의 존재**: 타입별 고대의 존재 카드
- **특수 고대의 존재**: 전설급과 신화급 특수 카드
- **희귀도별 분류**: 고대의 존재, 전설, 신화 등급
- **캐릭터별 추천**: 각 캐릭터에 최적화된 고대의 존재

## 기술 스택

- **HTML5**: 시맨틱 마크업과 구조화된 콘텐츠
- **CSS3**: 반응형 디자인과 현대적 스타일링
- **JavaScript**: 인터랙티브 기능과 동적 콘텐츠
- **Google Fonts**: 한국어 폰트 지원 (Noto Sans KR)

## 주요 특징

### 🎨 디자인
- **다크 테마**: 게임 분위기에 맞는 다크 테마
- **반응형 디자인**: 모든 기기에서 최적화된 화면
- **한국어 지원**: 완벽한 한국어 지원과 폰트
- **직관적 내비게이션**: 쉽고 빠른 정보 접근

### ⚡ 기능
- **실시간 필터링**: 카드, 유물, 적 정보 실시간 필터링
- **상세 모달**: 클릭하여 상세 정보 확인
- **부드러운 애니메이션**: 사용자 경험 향상
- **검색 기능**: 빠른 정보 검색 (추가 예정)

### 📱 사용자 경험
- **빠른 로딩**: 최적화된 코드로 빠른 로딩 속도
- **직관적 UI**: 쉽게 이해할 수 있는 인터페이스
- **상세한 정보**: 게이머에게 필요한 모든 정보 제공
- **전략 가이드**: 각 콘텐츠별 전략 정보 제공

**SpireArchive** - 슬레이 더 스파이어2의 완벽한 동반자

**프롬프트**
넌 이제부터 웹프로그래밍 전문가야

슬레이 더 스파이어2 라는 게임의 모든 정보를 담은 웹페이지를 만들 거고, 웹 페이지에 포함되어야 할 핵심 정보들은 캐릭터, 캐릭터 별 카드, 무색 카드, 유물, 이벤트 정보, 적 정보야.

핵심 정보: 캐릭터(총 다섯 종: 아이언클래드, 사일런트, 리젠트, 네크로바인더, 디펙트), 캐릭터 별 카드, 무색 카드, 유물, 이벤트 정보, 적 정보

https://slaythespire.wiki.gg/wiki/Slay_the_Spire_2:Main 의 사이트의 구조를 참고해서 만들어줘, 메인 페이지에서 캐릭터를 클릭하면 캐릭터를 상세히 소개하는 페이지로 연결되어야 하고, 이를 메인 페이지에 포함된 모든 핵심 정보들에 대해 똑같이 적용해야 해 

이미지가 들어가야 할 공간은 비워두고, 텍스트가 들어가야 할 자리에는 위 사이트에서 가져온 텍스트를 정확한 위치에 대응시켜줘, 이미지, 텍스트가 들어가야 할 공간은 수정하기 편하도록 주석 달아줘

그리고 현재 폴더 내에 웹페이지를 개발할 때 사용되는 통상적인 프로젝트 폴더 구조로 구성해 생성해줘

사용자층은 한국인 게이머기 때문에 웹페이지는 기본적으로 한국어로 작성되어야 해, 그리고 웹페이지의 이름은 'SpireArchive'이고 url은 spirearchive.gg로 할 거야.

**개선1**

현재 진행된 부분에서 '포션', '고대의 존재' 항목을 핵심 정보에 추가하고 캐릭터에 들어가면 나오는 '추천 유물'란을 '전용 유물'로 바꿔줘

그리고 https://namu.wiki/w/Slay%20the%20Spire%202/%EC%95%84%EC%9D%B4%EC%96%B8%ED%81%B4%EB%9E%98%EB%93%9C 사이트를 참고해서

캐릭터 '아이언클래드'와 '카드' 정보를 위 사이트와 동일하게 수정해줘

**개선2**

지금부터 CSS를 수정할거야

메인페이지의 카드, 유물, 이벤트를 같은 행에 배치하고, 포션, 적, 고대의 존재를 같은 행에 배치하는 구조로 해서 공간의 낭비를 줄여줘

그리고 각 항목엔 예시 이미지가 들어갈 수 있도록 공간을 확보해주고

캐릭터 항목은 수정할 필요 없어

**개선3**

메인페이지의 카드 항목의 '캐릭터별 카드' 버튼 밑의 '무색 카드' 버튼을 없애고, 캐릭터별 카드로 들어가면 나오는 캐릭터별로 구별해놓은 행 제일 오른쪽에다 '중립'을 같은 스타일 같은 기능으로 추가하고 '중립' 버튼을 누르면 캐릭터별 카드 안의 캐릭터를 클릭했을 때 제공되는 기능과 동일하도록 수정해줘

**개선4**


캐릭터별 카드 페이지 필터에서 중립 항목 삭제하고 character-cards.html 페이지에서 colorless-cards.html로 연결되도록 하는 '중립' 버튼이 나타나도록

            <div class="character-tabs">
                <button class="tab-btn active" data-character="ironclad">아이언클래드</button>
                <button class="tab-btn" data-character="silent">사일런트</button>
                <button class="tab-btn" data-character="regent">리젠트</button>
                <button class="tab-btn" data-character="necrobinder">네크로바인더</button>
                <button class="tab-btn" data-character="defect">디펙트</button>
            </div>

위 구조에 추가해줘

**개선5**

카드, 유물, 이벤트, 적 항목을 생성했었던 것처럼

메인 페이지의 모든 포션 보기, 모든 고대의 존재 보기 버튼을 클릭하면 연결되는 html파일과 그에 맞는 스타일, 기능들까지 만들어줘
